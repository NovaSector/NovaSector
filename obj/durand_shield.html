durand_shield
/obj/durand_shield

An object to take the hit for us when using the Durand's defense mode. It is spawned in during the durand's initilization. Normally invisible, until defense mode is actvated. When the durand detects an attack that should be blocked, the attack is passed to the shield. The shield takes the damage, uses it to calculate charge cost, and then sets its own integrity back to max. Shield is automatically dropped if we run out of power or the user gets out.
