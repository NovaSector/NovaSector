Title: Cargo Items

MODULE ID: CARGO_ITEMS

Description:

Cargo Teleporter: An item that can teleport items to a specified location marked by the items. It can be upgraded to do various additional things

Conveyor Sorter: An item that will sort items that cross over it. An item is used to obtain a list that can be inserted into the machine

Goody Case Holder: An item that will store 8 goody cases for easy access and easy delivery
